## Русский
 
### Что это
 
Этот репозиторий автоматически конвертирует списки доменов из проекта [v2fly/domain-list-community](https://github.com/v2fly/domain-list-community) в формат правил для [Shadowrocket](https://apps.apple.com/app/shadowrocket/id932747118).
 
Списки обновляются автоматически каждый день через GitHub Actions.
 
### Источник данных
 
Все данные берутся из репозитория [v2fly/domain-list-community](https://github.com/v2fly/domain-list-community) — открытого проекта сообщества, содержащего классифицированные списки доменов для работы с прокси и VPN.

### Как использовать
 
1. Откройте Shadowrocket
2. Перейдите в **Config → Rule Sets → +**
3. Вставьте URL нужного списка:
```
https://raw.githubusercontent.com/mahlenko/shadowrocket-v2fly/main/output/<name>.list
```

## English
 
### What is this
 
This repository automatically converts domain lists from [v2fly/domain-list-community](https://github.com/v2fly/domain-list-community) into rule files compatible with [Shadowrocket](https://apps.apple.com/app/shadowrocket/id932747118).
 
Lists are updated daily via GitHub Actions.
 
### Data source
 
All data comes from [v2fly/domain-list-community](https://github.com/v2fly/domain-list-community) — an open community project containing classified domain lists for use with proxies and VPN clients.

### How to use
 
1. Open Shadowrocket
2. Go to **Config → Rule Sets → +**
3. Paste the URL of the desired list:
```
https://raw.githubusercontent.com/mahlenko/shadowrocket-v2fly/main/output/<name>.list
```
